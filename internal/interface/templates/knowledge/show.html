¿Qué es la Base de Conocimiento?

¿Para qué sirve?

Es un repositorio de documentación técnica y de negocio que alimenta directamente a Claude cuando generás prompts para crear casos de prueba. Cuanto más completo esté, mejores serán los casos generados.

BUSINESS_KNOWLEDGE.md

Se regenera automáticamente cada vez que creás o eliminás un caso de prueba. Contiene un resumen de todos los casos existentes agrupados por reporte para dar contexto a Claude.

Documentos manuales

Podés agregar documentos propios con reglas de negocio, restricciones, flujos especiales o cualquier contexto que el equipo necesite documentar y compartir con Claude.

Flujo con Claude

En Reportes → Prompt Claude se genera un prompt que incluye todo el contenido de esta base. Copiás el prompt, lo pegás en Claude y obtenés casos de prueba listos para importar.
